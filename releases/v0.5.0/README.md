# 版本 0.5.0 发布信息

## 基本信息
- **版本号**: 0.5.0
- **发布日期**: 2026-02-11T13:00:00Z
- **发布类型**: minor（交互重构）
- **应用名称**: 个人目标管理 App
- **技术栈**: Expo 54 + React Native + Express + PostgreSQL

## 版本描述
重构目标详情页面，将创建目标和编辑目标合并为统一的表单页面，删除编辑Modal，采用创建模式的交互风格。

## 核心改动

### 功能改进
1. **统一创建和编辑体验**
   - 移除编辑目标的 Modal 弹窗
   - 创建和编辑使用相同的表单样式
   - 点击编辑按钮直接切换到编辑模式

2. **新增编辑模式**
   - Header 标题动态变化：创建目标 / 编辑目标 / 目标详情
   - 编辑模式显示"取消"和"保存"两个按钮
   - 取消编辑恢复原始数据

3. **交互优化**
   - 添加 KeyboardAvoidingView，自动处理键盘遮挡
   - 页面切换更流畅，无需等待 Modal 弹窗
   - 界面更简洁，操作更直观

### 修改文件
- `client/screens/goal-detail/index.tsx` - 页面逻辑重构
- `client/screens/goal-detail/styles.ts` - 样式更新，移除Modal相关样式

### 样式变更
- 新增 `editButtonGroup` - 编辑按钮组容器
- 新增 `cancelEditButton` - 取消按钮样式
- 新增 `saveEditButton` - 保存按钮样式
- 删除所有 Modal 相关样式（modalOverlay, modalContainer, modalContent 等）
